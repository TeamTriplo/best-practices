Welcome to Triplo Best Practices.

This is an experimental project to help Triplo team members practice/develop their git skills while updating a simple html document. While we are primarily a Drupal shop, we're keeping this document in basic html to keep the focus on GIT.

This project is findable - but we are not yet sharing it publicly.

Initially, this page will simply link to a Google Doc that we're using to maintain our best practices. Over time, we'll move that content into this html site and share it with the world.

Site Building Best Practices

NOTE: These are best practices as currently being used by Triplo. If you disagree with any of these practices, we welcome your comments.
